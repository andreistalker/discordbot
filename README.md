# Discord Bot

Discord bot is a simple bot for Discord made for my server. Feel free to use it too!
## Installation

Use [node.js](https://nodejs.org/en/) to install this bot.

1. Edit the config.json file.
2. ```
   npm install
   node index
   ```
3. Add the bot to your discord server!

#### You're all set now!
